# Taxis v0.0.1

## 📜 Description

Taxis is a very simple data structure library. Its craeted intended to serve as a dependency for the [**Physis**](https://github.com/hayashi19/Physis) package, providing necessary structural components for that project.

## 🧩 Components
- [x] Node
- [x] Tree
- [ ] Graph

## 💾 Installation

You can install Taxis directly from its GitHub repository using `pip`:

```bash
pip install git+https://github.com/hayashi19/Taxis.git
```
